JoyStock

Stage développeur full-stack • SaaS de gestion de stock (production)
Projet phare
FastAPIReactReact Native SupabasePostgreSQLJWT

Application SaaS complète de gestion de stock et d'inventaire, développée en stage full-stack pour Enjoy Tacos (franchise Enjo La Doua, Villeurbanne) et déployée en production pour un usage réel par l'équipe du restaurant.

Voir détails (fonctionnalités & points techniques)
  • Backend FastAPI (Render) + frontend React/Vite (Vercel) + app mobile React Native/Expo.
  • Base Supabase (PostgreSQL) avec Row Level Security, authentification JWT (ES256/JWKS).
  • Gestion des employés avec rôles/permissions (RBAC) et suppression logique (soft delete).
  • Prévisions de stock via fonctions SQL, module coûts & marges avec visualisations (Recharts).
  • Export CSV, envoi d'emails transactionnels (Resend) sur domaine dédié.
  • Suites de tests automatisés (pytest côté backend, Vitest côté frontend).
